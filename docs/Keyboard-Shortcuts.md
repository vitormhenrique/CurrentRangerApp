# Keyboard Shortcuts

## Chart

| Key | Condition | Action |
|-----|-----------|--------|
| **M** | Hovering chart | Add a point marker at the cursor position |
| **M** | Drag-selection active | Add a range marker spanning the selection |

## Marker Popup

| Key | Action |
|-----|--------|
| **Enter** | Confirm (add or save marker) |
| **Escape** | Cancel and close popup |

## Marker Panel (Edit Mode)

| Key | Action |
|-----|--------|
| **Enter** | Save changes |
| **Escape** | Cancel editing |
